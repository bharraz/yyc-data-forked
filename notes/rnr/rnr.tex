\section{Quantum model}

The initial state of the system (thermal) and the time evolution of it in the harmonic trap
as well as free space can both be easily described by the Wigner function.
After obtaining the Wigner function of the final state after recapturing, we will assume that
the escape probability is the total distribution in the phase space that has too high energy.
We also need to consider the 3D case, where the atom can escape as long as the total energy
in all dimension exceeds the trap depth.

\section{Initial state}

Define dimensionless quantities,

\eqar{
  x=&\sqrt{\frac{m\omega}{\hbar}}x_0\\
  p=&\frac{p_0}{\sqrt{m\hbar\omega}}\\
  t=&\omega t_0\\
  \beta=&\beta_0\hbar\omega
}

where $x_0$, $p_0$, $t_0$, etc. are the original dimensional quantities
and $x$, $p$, $t$, etc. are dimensionless.

In this unit, the energy is

\eqar{
  E=&\frac{m\omega^2x_0^2}{2}+\frac{p_0^2}{2m}\\
  =&\frac{\hbar\omega}{2}\paren{x^2 + p^2}\\
  \equiv&\hbar\omega\varepsilon
}

For the initial thermal state at temperature $T$, the (dimensionless) Wigner function is

\eqar{
  W(x,p)=&\frac{\tanh(f)}{\pi}\exp\paren{-\tanh(f)\paren{x^2 + p^2}}
}
where $f\equiv\dfrac{1}{2}\beta$.

\section{Evolution}
The evolution of the Wigner function in free space is a shearing along $x$.
After time $t$, the distribution $W_0(x,p)$ becomes,
\eqar{
  W_t(x,p)=&W_t(x-pt,p)
}

For the thermal initial state, this is
\eqar{
  W_t(x,p)=&\frac{\tanh(f)}{\pi}\exp\paren{-\tanh(f)\paren{(x-pt)^2 + p^2}}\\
  =&\frac{\tanh(f)}{\pi}\exp\paren{-\tanh(f)\paren{x^2-2txp+(1 + t^2)p^2}}
}

\section{Energy distribution}

To get the energy distribution after the recapture, we can integrate the Wigner function over
the polar angle.

\eqar{
  \diff{P}{r}=&\int_0^{2\pi}\ud\theta rW_t(r\cos\theta, r\sin\theta)\\
  =&r\frac{\tanh(f)}{\pi}\int_0^{2\pi}\ud\theta\exp\paren{-\tanh(f)r^2\paren{\cos^2\theta-2t\cos\theta\sin\theta+(1 + t^2)\sin^2\theta}}\\
  =&r\frac{\tanh(f)}{\pi}\ue^{-\tanh(f)\paren{2+t^2}\varepsilon}\int_0^{2\pi}\ud\theta
  \exp\paren{\tanh(f)\varepsilon\paren{t^2\cos2\theta+2t\sin2\theta}}
  \intertext{Energy distribution}
  \diff{P}{\varepsilon}=&\frac{\tanh(f)}{\pi}\ue^{-\tanh(f)\paren{2+t^2}\varepsilon}\int_0^{2\pi}\ud\theta
  \exp\paren{\tanh(f)\varepsilon\paren{t^2\cos2\theta+2t\sin2\theta}}
  \intertext{Shift $\theta$}
  \diff{P}{\varepsilon}=&\frac{\tanh(f)}{\pi}\ue^{-\tanh(f)\paren{2+t^2}\varepsilon}\int_0^{2\pi}\ud\theta\exp\paren{\tanh(f)t\sqrt{4+t^2}\varepsilon\cos2\theta}\\
  =&2\tanh(f)\ue^{-\tanh(f)\paren{2+t^2}\varepsilon}I_0\paren{\tanh(f)t\sqrt{4+t^2}\varepsilon}
}
where $I_n(x)$ is the modified Bessel function of the first kind.
Define
\eqar{
  G\paren{x, t}\equiv&2\ue^{-\paren{2+t^2}x}I_0\paren{t\sqrt{4+t^2}x}\\
  \diff{P}{\varepsilon}=&\tanh\paren{\frac{\beta}{2}}G\paren{\varepsilon\tanh\paren{\frac{\beta}{2}}, t}
}
